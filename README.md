# 考研复习管理系统

一个纯前端的考研复习管理工具，支持错题本、学习计划、知识点热力图、番茄钟、学习进度对比和 Forest 数据导入。

现在支持两种使用方式：

- 本地模式：直接打开 `index.html`，数据只保存在当前浏览器。
- WiFi 共享模式：一台电脑运行共享服务，两个人在同一 WiFi 下访问同一个地址，共用同一份数据。

## 项目结构

```text
.
├── index.html              # 页面结构和入口
├── server.js               # WiFi 共享服务
├── start-wifi-share.bat    # Windows 一键启动脚本
├── start-wifi-share.ps1    # Windows PowerShell 启动脚本
├── package.json            # Node 启动配置
├── data
│   └── app-data.json       # 运行后自动生成的共享数据文件
└── assets
    ├── css
    │   └── app.css         # 页面样式
    └── js
        └── app.js          # 应用逻辑和同步管理
```

## 使用方式

直接用浏览器打开 `index.html` 即可运行。数据保存在浏览器本地存储中，导入/导出功能可用于备份和迁移。

## WiFi 共享模式

1. 在主电脑上双击 `start-wifi-share.bat`。
2. 窗口会显示几个访问地址，例如 `http://192.168.1.23:8787`。
3. 主电脑和另一台设备连接同一个 WiFi。
4. 两个人都用浏览器打开同一个 `http://局域网IP:8787` 地址。
5. 页面左下角显示 `WiFi共享模式` 时，说明正在共用同一份数据。

共享数据会保存在主电脑的 `data/app-data.json`。运行时不要关闭启动窗口；关闭后其他设备就不能继续访问。
